phantom-testlib
by shoptime

Library for testing websites, backed by phantomjs, TAP compatible.

A library for writing web application tests in javascript. Provides a procedural interface for writing tests, so you don't have to grok event based programming. Uses PhantomJS to provide a webkit-based headless browser to run your tests against.

Synopsis

phantom.injectJs('testlib.js');
var t = new Test('https://github.com');
t.open('/');
t.is(t.text('title'), 'GitHub Â· Social Coding', 'Homepage loaded');
t.done();

Install

  1. Clone the repository
  2. Install PhantomJS: 
    # Ubuntu instructions
sudo add-apt-repository ppa:jerome-etienne/neoip
sudo apt-get update
sudo apt-get install phantomjs
# See http://code.google.com/p/phantomjs/wiki/Installation for more
  3. Test: 
    phantomjs github-test.js
