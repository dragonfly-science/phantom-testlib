Class Detail
Test(base_url)
A library for writing web application tests in javascript. Provides a procedural interface for writing tests, so you don't have to grok event based programming. Uses PhantomJS to provide a webkit-based headless browser to run your tests against. Synopsis: 
phantom.injectJs('testlib.js');
var t = new Test('https://github.com');
t.open('/');
t.is(t.text('title'), 'GitHub Â· Social Coding', 'Homepage loaded');
t.done();
Parameters:
{String} base_url
The base URL that all #open calls will be relative to.

click(selector)
Sends a click event to the element matching the given selector. If the element is a link, it will be followed, and if the element triggers a form submission, the form will be submitted. If you're expecting the click to take the browser to another page, consider using #click_and_wait instead.
Parameters:
{String} selector
The element to click on.

done()
Tell the testlib that you are done with your tests. This will output a TAP summary of the tests run, and exit. Calling this lets TAP parsers know that your script finished (as opposed to crashing), so make sure you call it.
is(got, expected, description)
Check that the one value matches another. Typically, you would call this method like so: 
t.is(t.text('title'), 'My Title', 'Page title is correct');
Parameters:
got
The value to check - typically you should use the result of a call to #text, #val, #attr or similar here.
expected
The value you expect. Will be compared using ==.
{String} description
The test description.

sleep(ms)
Pauses the test execution. Note that this has many legitimate uses, but you shouldn't use it if there is a 'wait' method available that can do the job for you, as they return as soon as the event they're waiting for has happened. 
t.sleep(1000); // sleep for a second
Parameters:
{Integer} ms
How many milliseconds to sleep for.
